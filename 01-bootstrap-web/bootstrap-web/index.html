Tlačítka

Bootstrap umožňuje snadné stylování tlačítek pomocí tříd jako např. btn-primary



            
                <button class="btn btn-primary">Primary button</button>
                <button class="btn btn-warning">Warning button</button>

Karty

Karta v Bootstrapu je super způsob jak strukturovat obsah.


Ukázková karta

Obsah ukázkové karty.

Detail 

            
                <div class="card-body">
                    <h3 class="card-title">Ukázková karta</h3>
                    <p class="card-text">Obsah ukázkové karty.</p>
                    <a href="#" class="btn btn-info btn-sm">Detail</a>
                </div>

Tabulky

Tabulky lze v Bootstrapu stylovat pomocí tříd jako např. table-striped
